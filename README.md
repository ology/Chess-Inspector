Chess::Inspector
====

Visualize move, protection and threat status
----

*Installation*

    git clone https://github.com/ology/Chess-Inspector.git
    cd Chess-Inspector
    cpanm --installdeps .
    plackup bin/app.pl

Now browse to http://localhost:5000/
